Welcome to Matrix Throttle

Project overview

Matrix throttle will limit the number of child jobs launched simultaneous.

Motivation

Consider an environment with several projects but a limited number of licenses for certain tools. If a single commit spawns 100 jobs and you can run 10 of them in parallel? With the default jenkins implementation you were had the option of running them in sequence or clogging up all execution nodes with jobs waiting for the a licence. With all jobs aggressively asking for a key to run, manuel users will be stuck in queue for hours until it is their turn. The Matrix-throttle jenkins addon will limit the number of child jobs submitted to the executor
